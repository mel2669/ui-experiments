
  # Hover Effects Showcase

  This is a code bundle for Hover Effects Showcase. The original project is available at https://www.figma.com/design/1ALhfjwgkaPNdcyvputpdP/Hover-Effects-Showcase.

  ## Running the code

  Use Node.js 18+ (LTS recommended). From this folder:

  - `npm install` — install dependencies
  - `npm run dev` — start the Vite dev server (edit files under `src/`; the browser hot-reloads)
  - `npm run build` — production build to `dist/`
  - `npm run typecheck` — TypeScript check (same config the editor uses)

  If you use pnpm, `pnpm install` and `pnpm dev` work the same (`pnpm-workspace.yaml` is only for the root package).
